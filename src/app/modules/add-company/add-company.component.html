Steps for Importing a New Company:

1. Download the PDF version of annual financial statement of the company (10-K)

2. Edit the PDF file to remove all pages except for the Income Statement, Balance Sheet, and Cash Flow Statements

3. Upload the edited file by either dragging it into the box to the right or using the Browse option

4. Populate the fields listed above as it relates to key company attributes (e.g., Company name, Industry, etc.)
